gxkb

X11 keyboard indicator and switcher

Download as .zip Download as .tar.gz View on GitHub

Description

It shows a flag of current keyboard in a systray area and allows you to switch to another one. It's written in C and uses the GTK library (no GNOME is needed)

Having trouble with gxkb? Please send bug report to the GitHub Issue Tracker. Feel free to use this issue tracker to contact @zen-tools with questions and comments.
